CSS Grid

2019-10-20T12:46:37.121Z

CSS Grid is a gamechanger. It's about damn time there's a simple way to create grids without requiring ugly hacks.

I'm not a fan of long tutorials that waste your time, so let's get to the point.

This is a grid with 3 columns:

HTML

<div class="grid-container">
  <div>Thing 1</div>
  <div>Thing 2</div>
  <div>Thing 3</div>
  <div>Thing 4</div>
  <div>Thing 5</div>
  <div>Thing 6</div>
  <div>Thing 7</div>
</div>

CSS

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  /* grid-template-columns: repeat(3, 1fr); Does same as above */
  grid-gap: 10px;
  grid-auto-rows: minmax(100px, auto);
}

#container > div {
  background-color: hsl(0,0%,10%);
}

That's it! End of tutorial.
