Unix Commands / Shortcuts

2019-08-20T17:21:33.962Z
Shortcut Command
Ctrl+L Clear Screen
Ctrl+R Reverse-i Search

More:

  • https://skorks.com/2009/09/bash-shortcuts-for-maximum-productivity/

View a file:

  • <file.txt
  • more file.txt
  • tail file.txt
  • cat file.txt

Search a file

  • <file.txt grep searchquery
  • grep searchquery file.txt
  • cat file.txt | grep searchquery

Write to a file:

  • echo "Content of file" > file.txt

Multiline:

cat << EOF > file.txt
First line of file
Second line of file
Wow multiple lines, sweet
EOF

Linking:

  • ln -s <link_destination)

Permissions:

  • sudo chown -R $(whoami) /usr/local/lib

Sending files remotely

  • scp
  • rsync

View processes / Task Manager

  • ps -ef
  • uptime
  • top
  • htop
  • free -m

System info

  • cat /proc/cpuinfo

Disk space

  • df -h
  • du -h

Generating SSH Key

  • ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
  • eval $(ssh-agent -s)
  • ssh-add ~/.ssh/id_rsa

Multiple sessions / Tabs / Split pane

  • tmux

Help

  • man [command]
